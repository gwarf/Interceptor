Interceptor (Browser-Only) is installed.

1Load or reload the browser extension

Chrome and Brave do not allow installers to load or refresh extensions automatically — do this once by hand, and after package updates:

  1. Open chrome://extensions/ (or brave://extensions/)
  2. Toggle Developer mode on (top right)
  3. Click Load unpacked
  4. Select this folder:
    /Library/Application Support/Interceptor/extension
  5. If Interceptor was already loaded from this folder, click its Reload button instead.

After the extension is connected, interceptor reload also refreshes the running extension service worker.

2Optional: register skill packs with your AI tooling

Interceptor ships skill packs that teach AI agents how to drive the browser surface and how to research deeply (the interceptor-research methodology skill). One command links the skills you choose into the AI runtimes it detects (Claude Code, Codex, and any ~/.agents/skills consumer) — and keeps them current across updates:

interceptor skills adopt

Check what is linked at any time with interceptor skills status. Prefer to do it by hand? Symlink any skill directly, e.g. for Claude Code:

mkdir -p ~/.claude/skills
ln -sfn "/Library/Application Support/Interceptor/skills/interceptor-browser" ~/.claude/skills/interceptor-browser

The same pattern works for any runtime that reads a skills folder (~/.codex/skills, ~/.agents/skills, ~/.openclaw/skills, ~/.config/opencode/skills).

Try it

Open a terminal and run:

interceptor open "https://example.com"

To uninstall later, run
sudo bash "/Library/Application Support/Interceptor/uninstall.sh"
